# Next.js 프로젝트 <중고마켓>

패스트캠퍼스 강의를 보고 공부한 소스코드 입니다.

## 작업내용

- UI 컴포넌트 생성
- Next Auth 설정
- 권한에 따른 인가 처리
- 로그인한 사람만 특정 경로 접근가능
- 디비 스키마 생성 및 설정
- 회원가입 기능 구현
- 해시비밀번호를 이용한 비밀번호 암호화
- 로그인시 유저 세션 데이터 생성
- next-cloudinary 이미지 업로드

## 개발환경

- HTML, Tailwind CSS, TypeScript, Next.js
- React-hook-from, bcrypt, Auth.js, NextAuth.js, react-icons, next-cloudinary
- Prisma, Postgres, Docker
